🏋️ AI Fitness Trainer – Pose Estimation

An AI-based fitness trainer that uses computer vision to detect body posture, track exercises, and count repetitions in real time using a webcam.

Built with Python, Google MediaPipe, and OpenCV.

🚀 Features

Real-time pose detection

Automatic rep counting

Joint angle calculation

Posture feedback

Supports multiple exercises

🏋️ Supported Exercises

Squats

Hammer Curls

Push-Ups

🛠️ Tech Stack

Python

MediaPipe

OpenCV

NumPy

▶️ How to Run
pip install mediapipe opencv-python numpy
python main.py

Make sure your webcam is connected.
Press Q to exit.

🎯 Objective

To build a virtual AI fitness trainer that monitors workout posture and counts repetitions accurately in real time.
